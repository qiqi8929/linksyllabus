Homepage hero demo video (not committed — too large for Git)
=============================================================

1) Supabase Storage
   - Dashboard → Storage → New bucket (e.g. name: marketing) → Public bucket ON.
   - Upload your file as: hero-demo.mp4 (or any path you prefer).

2) Public URL (copy from file details after upload)
   Shape:
   https://<PROJECT_REF>.supabase.co/storage/v1/object/public/<BUCKET>/hero-demo.mp4

3) Environment variable (local + Vercel)
   In .env.local (local) and Vercel → Project → Settings → Environment Variables:

   NEXT_PUBLIC_LANDING_HERO_DEMO_MP4=https://<PROJECT_REF>.supabase.co/storage/v1/object/public/<BUCKET>/hero-demo.mp4

   Restart dev server after changing .env.local.

4) Git
   public/hero-demo.mp4 is listed in .gitignore — do not commit the MP4.

The homepage default URL is set in src/app/page.tsx (Supabase marketing bucket). Override with NEXT_PUBLIC_LANDING_HERO_DEMO_MP4 if needed.
